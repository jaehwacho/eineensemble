# 플루트 앙상블 소개 홈페이지 제작 계획

## Context
**아이네 앙상블 (EineEnsemble)** 플루트 앙상블 그룹을 소개하는 홈페이지를 새로 제작한다. 빈 디렉토리에서 시작하며, 클래식하고 우아한 디자인으로 그룹의 정체성을 표현한다. 실제 콘텐츠(이미지, 단원 정보 등)를 사용한다.

## 기술 스택
- **순수 HTML/CSS/JS** (빌드 도구 없음, 즉시 브라우저에서 실행 가능)
- Google Fonts CDN (Playfair Display + Nanum Myeongjo for 제목, Noto Sans KR for 본문)
- Formspree (서버 없는 이메일 폼, 무료)
- 외부 JS 라이브러리 없음 (Vanilla JS로 모든 인터랙션 구현)
- **다국어 지원**: 한국어(ko) / 영어(en) JS 기반 i18n (페이지 새로고침 없이 언어 전환)

## 브라우저 지원 범위
- Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- IE 미지원 (CSS `columns`, IntersectionObserver 등 모던 API 사용)
- 모바일: iOS Safari 14+, Android Chrome 90+

## 섹션 구성 (5개)

### 1. Hero
- 전체 화면(100vh), 공연 사진 배경 + 다크 오버레이
- **"아이네 앙상블 EineEnsemble"** (대형 세리프 타이포), 슬로건, CTA 버튼
- 텍스트 페이드인 + 슬라이드업 애니메이션
- 스크롤 유도 화살표(bounce)

### 2. About
- 2컬럼: 텍스트(창단 역사, 비전) + 단체 사진
- 통계 카운터: 창단 연도(1997), 정기연주회 횟수(32+), 단원 수(16명) (뷰포트 진입 시 카운팅 시작)
- 주요 해외 공연 국가 하이라이트 (오스트리아, 미국, 베트남)

### 3. Members
- **16명** 반응형 카드 그리드 (데스크톱 4열 → 태블릿 3열 → 모바일 2열)
- 카드: 프로필 사진, 이름, 포지션, 호버 시 오버레이
- 카드 클릭 → 개별 상세 프로필 페이지(`members/[name].html`)로 이동
- **단원 상세 페이지**: 대형 프로필 사진, 이름/포지션, 학력, 수상 경력, 활동 이력, 레퍼토리 등 상세 이력 전체 표시
- 상세 페이지 하단 "← 멤버 목록으로" 뒤로가기 버튼
- 데이터 소스: `data/members.json` fetch 후 렌더링 (콘텐츠 수정 시 JSON만 편집)

### 4. Gallery / 공연 일정
- 상단: 예정 공연 리스트 (날짜, 장소, 프로그램, 티켓 링크) — `data/concerts.json`
- 하단: 공연 사진 Masonry 그리드 + 클릭 시 라이트박스
- CSS `columns`로 Masonry 구현 (외부 라이브러리 불필요)

### 5. Contact
- 문의 폼 (이름, 이메일, 메시지) → Formspree 연동
- SNS 링크 아이콘 버튼: 인스타그램, **유튜브 채널 링크** (필수)
- Vanilla JS 폼 유효성 검사
- 주소 및 연락처 표시

## 디자인 시스템

### 디자인 방향: 클래식 플루트 앙상블

악보의 섬세한 선율, 콘서트홀의 고급스러움, 목관악기 특유의 따뜻하고 맑은 이미지를 시각 언어로 표현한다. 악보 오선지 모티프(subtle 배경 장식), 우아한 세리프 타이포, 따뜻한 아이보리 + 금색 팔레트 사용.

```css
:root {
  /* 클래식 음악 느낌: 아이보리 + 딥 버건디/포레스트 + 골드 */
  --color-primary: #2c1810;      /* 딥 브라운-버건디: 목관악기 나무 질감 */
  --color-secondary: #b8860b;    /* 다크 골드: 금빛 플루트 키 */
  --color-accent: #d4af37;       /* 밝은 골드 */
  --color-ivory: #f8f4e9;        /* 아이보리: 악보지 느낌 */
  --color-bg-dark: #1a0f08;      /* 콘서트홀 어두운 배경 */
  --color-bg-light: #fdfaf4;     /* 크림 밝은 배경 */
  --color-bg-warm: #f5ede0;      /* 따뜻한 베이지 섹션 */
  --color-text-muted: #7a6352;   /* 중간 갈색 보조 텍스트 */

  --font-heading: 'Playfair Display', 'Nanum Myeongjo', serif;
  --font-body: 'Lato', 'Noto Sans KR', sans-serif;

  --section-padding: clamp(4rem, 8vw, 8rem);
  --container-max: 1200px;

  /* 악보 오선지 모티프 (CSS 배경 패턴으로 subtle 적용) */
  --pattern-staff: repeating-linear-gradient(
    transparent 0px, transparent 10px,
    rgba(184, 134, 11, 0.06) 10px, rgba(184, 134, 11, 0.06) 11px
  );
}
```

## 핵심 기술 구현

| 기능 | 구현 방법 |
|------|----------|
| 스크롤 애니메이션 | IntersectionObserver (scroll 이벤트보다 성능 우수) |
| 카운터 | 뷰포트 진입 시 requestAnimationFrame 카운팅 |
| 모바일 메뉴 | JS 클래스 토글 + CSS transform 풀스크린 오버레이 |
| 라이트박스 | Vanilla JS (외부 라이브러리 없음) |
| 이미지 최적화 | `loading="lazy"`, `<picture>` + srcset, WebP |
| 이메일 폼 | Formspree (무료, 서버 불필요) |
| 접근성 | prefers-reduced-motion 대응, aria 속성 |
| **다국어(i18n)** | HTML `data-i18n` 속성 + `i18n.js` 언어 데이터 객체. 네비 우측에 KO/EN 토글 버튼. `localStorage`로 선택 언어 유지. `lang` 속성 변경으로 폰트 자동 전환. |
| 콘텐츠 관리 | `data/members.json`, `data/concerts.json` → JS fetch 렌더링. 추후 수정 시 JSON만 편집. |
| SEO | sitemap.xml, robots.txt, Open Graph 메타태그, JSON-LD 구조화 데이터 |
| 파비콘/PWA | favicon.svg + site.webmanifest (홈 화면 추가 지원) |

## 구현 순서

1. **데이터 구조 설계**: `data/members.json`, `data/concerts.json` 스키마 정의 (한/영 필드 포함)
2. `index.html` 시맨틱 마크업 + 5개 섹션 골격 + `data-i18n` 속성 삽입 + SEO 메타태그/OG태그
3. `style.css` CSS 변수(클래식 팔레트), 리셋, 기본 레이아웃
4. 섹션별 스타일링 (Hero → About → Members → Gallery → Contact)
5. `member-profile.css` + `members/template.html` → 16개 단원 상세 페이지 생성
6. `animations.css` 키프레임 정의
7. `i18n.js` ko/en 언어 데이터 객체, 언어 스위처 초기화
8. `navigation.js` 스크롤 헤더, 모바일 메뉴
9. `animations.js` IntersectionObserver
10. `gallery.js` 라이트박스
11. `main.js` 초기화 및 통합
12. `sitemap.xml`, `robots.txt`, `favicon`, `site.webmanifest` 생성
13. 반응형 점검, 이미지 최적화
14. 배포 설정 (GitHub Pages 또는 Netlify)

### 단체 소개 (About 섹션)

> 「아이네앙상블」은 사회적 소외계층과의 화합을 도모하며 1997년 경남 최초의 플루트 전문연주단체로 창단되어 병원과 사회복지시설 방문 연주로 재능기부와 다양한 문화 교류 활동을 하였습니다. 2020년, 「아이네앙상블」로 개명한 후 서양악기와 국악기의 합동공연을 통하여 오스트리아, 미국, 베트남 등 해외 순회공연과 워크샵으로 한국 문화와 예술을 알리며, '조각가 문신'과의 협업 프로젝트를 시행하고 미술, 영상, 무용과 협업하며 융복합 연주단체로서의 활동을 확장하고 있습니다.
>
> 「아이네앙상블」은 한국의 예술과 문화를 국내외에 널리 알리는 데 기여하고 민간외교관으로 활동하고 있으며, 각 공연마다 다른 주제를 담아 예술을 통하여 메시지를 전달하는 연주단체로 활동합니다.

- **메인 페이지(index.html)**: 전체 KO/EN 전환 지원
- **단원 상세 페이지(members/\*.html)**: `data/members.json`의 ko/en 필드 활용하여 동일한 KO/EN 전환 지원
